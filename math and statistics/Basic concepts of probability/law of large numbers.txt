Let's write a Python function that demonstrates the Law of Large Numbers.

Function Signature:

import random

def law_of_large_numbers(n: int) -> float:
    total_sum = 0

    for _ in range(n):
        random_number = random.random()
        total_sum += random_number

    average = total_sum / n
    return average
